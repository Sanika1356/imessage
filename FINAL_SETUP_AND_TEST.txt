===============================================
🐛 IF SOMETHING DOESN'T WORK
===============================================

Problem 1: "Can't send messages"
Fix:
- Check BOTH terminals are running
- Check MongoDB is connected (see backend terminal)
- Check browser console for errors (Press F12)
- Try refreshing the page

Problem 2: "User not found"
Fix:
- Make sure BOTH users have added phone numbers
- Include country code (+1, +91, etc.)
- Try searching by email instead
- Check both users are logged in

Problem 3: "Connection error"
Fix:
- Restart backend: Ctrl+C, then npm run dev
- Restart frontend: Ctrl+C, then npm run dev
- Clear browser cache: Ctrl+Shift+Delete
- Try different browser

Problem 4: "Messages not appearing"
Fix:
- Check Socket.IO connection (F12 → Console)
- Look for "Socket connected" message
- Refresh both browser windows
- Check firewall/antivirus isn't blocking

Problem 5: "Black screen after login"
Fix:
- Open browser console (F12)
- Look for error messages
- Check if Clerk webhook is configured
- Try signing out and in again
- Clear cookies and cache
