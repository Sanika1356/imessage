🔍 HOW TO ADD PHONE NUMBER:
----------------------------
If you don't see the phone prompt:

1. Click green (+) icon in top right
2. You'll be prompted to add your phone
3. Enter with country code: +[country code][number]
   Examples:
   - USA: +12345678901
   - India: +919876543210
   - UK: +447123456789
   - Canada: +16475551234

4. Click "Save Phone Number"
5. Now you can find contacts!

🔎 HOW TO FIND CONTACTS:

🐛 TROUBLESHOOTING:
-------------------

Problem: "Can't find user"
Solution: 
- Check phone number has country code (+1, +91, etc.)
- Verify they've added their phone number
- Try searching by email instead

Problem: "Messages not sending"
Solution:
- Check both backend and frontend are running
- Check browser console for errors (F12)
- Verify MongoDB is connected
- Check Clerk webhook is configured

Problem: "User profile not synced"
Solution:
- Sign out and sign in again
- Check Clerk webhook is working
- Check backend logs for errors

Problem: "Contact sync not working"
Solution:
- Only works on Chrome for Android
- Grant permissions when prompted
- Use manual add on desktop

🔧 QUICK FIXES:
---------------

1. Clear Cache:
   - Ctrl+Shift+Delete (Chrome)
   - Clear cookies and cache
   - Refresh page

2. Restart Servers:
   - Stop both terminals (Ctrl+C)

   - Press F12 in browser
